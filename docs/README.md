# Documentation Directory

This directory contains living documentation for the Moltbook research hackathon.

## Files

- **[FINDINGS.md](FINDINGS.md)**: Catalog of research findings as we discover them. Each finding includes visualization, methods, stats, and interpretation.

- **[PLATFORMS.md](PLATFORMS.md)**: Survey of agentic social network platforms (Moltbook, WikiMolt, ClawWork, Agents of Chaos). Platform features, API access, comparative opportunities.

- **[RESEARCH_QUESTIONS.md](RESEARCH_QUESTIONS.md)**: Organized catalog of potential research questions across 6 categories. Tracks status (brainstorming → planned → in progress → complete).

- **[TWITTER_THREAD_TEMPLATE.md](TWITTER_THREAD_TEMPLATE.md)**: Template and examples for sharing findings on Twitter in Papaillopoulos style. Includes visualization best practices.

## Workflow

1. **Before starting analysis**: Review `RESEARCH_QUESTIONS.md`, update status
2. **During analysis**: Take notes, draft visualizations
3. **After completing finding**: Add to `FINDINGS.md` with full details
4. **Before sharing**: Draft Twitter thread using template
5. **Continuously**: Update `PLATFORMS.md` as we learn about other networks

## Status Tracking

Use emoji status indicators:
- 🔵 Brainstorming
- 🟡 Planned
- 🟢 In Progress
- ✅ Complete
- 🔴 Blocked
- ⚫ Deprioritized

## Contributing

When adding a finding to `FINDINGS.md`:
1. Use the provided template
2. Include all required sections (punchline, methods, stats, code location)
3. Add high-quality visualization (300+ DPI)
4. Document limitations honestly
5. Link to replication code
