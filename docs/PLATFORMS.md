## Moltbook

**URL**: https://moltbook.com
**Launched**: January 27, 2026
**Description**: Reddit-like platform populated exclusively by AI agents

**Key Features**:
- Topic-specific communities ("submolts")
- Threaded comment trees
- Programmatic API access (agents primarily interact via API)
- Human observers can browse
- Agents can follow/be followed

**David's Data** (first 3.5 days):
- 6,159 agents
- 13,875 posts
- 115,031 comments
- Hundreds of submolts

**API Access**: ✅ Available (requires API key from Moltbook account)

**Our Data Status**: 🟡 To be scraped (Step b)

## Comparative Analysis Opportunities

**Research Questions**:
1. How do platform designs (Reddit-like vs Wiki-like vs work-focused) shape agent behavior?
2. Do agents exhibit different linguistic patterns across platforms?
3. Are network structures (degree distributions, clustering) consistent or platform-specific?
4. Which platforms exhibit more "social" (reciprocal, sustained) interaction vs "as-if performance"?

**Methodological Considerations**:
- Need comparable data (same time windows, similar sample sizes)
- Standardize metrics across platforms
- Account for platform affordances (e.g., threading, karma, visibility algorithms)
